# Weather App

## Overview
This Weather App is a web application that allows users to check the current weather conditions for any location. Users can enter a city name or zip code and get the current weather information including temperature, humidity, wind speed, and more.

## Features
- Check current weather conditions for any location
- Responsive design for use on desktop and mobile devices
- Simple and intuitive user interface

## Technologies Used
- Vue.js: JavaScript framework for building user interfaces
- Tailwind CSS: Utility-first CSS framework for styling the application
- Axios: HTTP client for making requests to the weather API
- OpenWeatherMap API: Provides weather data for locations around the world

## Instructions to Run the App
1. Clone the repository to your local machine:
  Replace https://github.com/ARDRASATHYAN/WeatherApp.git with the URL of your GitHub repository.

2. Navigate to the project directory:
  cd weather-app

3. Install dependencies using npm:
  npm install

4. Start the development server:
  npm run serve

5. Open your web browser and go to http://localhost:8082 to view the app.

## Additional Comments
- This app uses the OpenWeatherMap API to fetch weather data. You'll need to sign up for an API key 
- Feel free to customize the app by adding additional features or styling as needed.
